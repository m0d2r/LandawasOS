# LandawasOS
LandawasOS is an operating system for cc: tweaked

Features:
1. cleaner - Clear old update files   
2. Update - Updates the system to latest version
3. Program list - Shows Installed programs (system-only for now)
4. easy installer - Installs the system

# How to install
1. Run this in the terminal
```LUA
lua
```
3. them run this:

```LUA
shell.run("delete install.lua"); shell.run("wget https://raw.githubusercontent.com/m0d2r/LandawasOS/main/install.lua install.lua"); shell.run("install.lua")
```
3. done

This will install LandawasOS to the computer
